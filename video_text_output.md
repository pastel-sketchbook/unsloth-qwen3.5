## Second 137

직관적인 문장 구조: 'Pro-Drop' 현상

Both Spanish and Korean are 'pro-drop' languages, meaning they naturally omit subject pronouns when the context is clear. English requires a mandatory subject. This allows you to instinctively form more native-sounding sentences, avoiding the redundancy common among English-speaking beginners.

| | | |
| :--- | :--- | :--- |
| **Spanish (Natural)** | \*Voy a la tienda.* | **Korean (Natural)** | 가지에 가요. |
| **English (Mandatory Subject)** | I go to the store. |

Note: The literal translation Yo voy a la tienda is grammatically correct but often sounds unnatural, a common pitfall for learners translating directly from English.

*   *How* you study is as important as *what* you study. To overcome the natural *Forgiving Curve*, use a scientifically proven method to ensure new information moves into your long-term memory.
*   **The Concept:** 간격 반복 (간격 반복 학습법). This actively combats memory decay.
*   **The Method (The 2-3-5-7 Rule)**
    *   For a bi-weekly learning schedule, review new material...
    *   2 hours after the lesson (summary).
    *   3 days later (active recall).
    *   5 days later (contextual practice).
    *   7 days later (final review before the next lesson).

*   **The Core Principle:** Focus on Active Recall (동적적 회상)—testing yourself—rather than passive re-reading.
